# Solution to task 1

*Assuming cwd is the RoFI.*

 1. Compile
    ```sh
    source setup.sh Debug
    rcfg desktop && rmake desktop
    ```

 2. Run the simulator
    ```sh
    source setup.sh Debug
    rofi-simplesim tutorials/simulator/task_1_solution/configuration.json
    ```
