Parsing Configurations in C++

When parsing configurations in C++ tools, the configurations get usually converted into rofi::configuration::RofiWorld. For this, tools can use the parsing function rofi::parsing::parseRofiWorld() or rofi::parsing::parseRofiWorldSeq().

For the other conversion, tools can use rofi::parsing::writeRofiWorld() or rofi::parsing::writeRofiWorldSeq().

Note

When parsing you may want to check whether the configuration is empty or whether the configuration is fixed in space.

If you want to allow for not fixed configurations and be able to prepare them (without regards to the real position), you can use rofi::parsing::fixateRofiWorld().

Also don’t forget to validate the configuration if needed.
