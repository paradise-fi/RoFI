Introduction, compiling and structure

Welcome to the RoFI project. In this chapter we will give you the basic overview on the sources & data organization in the repository. We will also show you, how to compile the project.

First, start by cloning the project repository:

$ git clone https://github.com/paradise-fi/RoFI.git
$ git clone git@github.com:paradise-fi/RoFI.git # If you have SSH keys set up
